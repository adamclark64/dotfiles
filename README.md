# Dotfiles for macos

I save my configuration files in this repository for an easy
setup of a new machine.

**Note: this is a work in progress.**

This repository includes configuration for:

- [Brew](https://brew.sh/)
- [Zsh](http://www.zsh.org/)
- [OhMyZsh](http://ohmyz.sh/)

## Usage

Run it with:

``` bash
bash bootstrap.sh
```
